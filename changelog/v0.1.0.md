## @globio/sdk v0.1.0

The first public release of the official Globio JavaScript/TypeScript SDK.

### What's included

**GlobioId** — Full authentication for your game. Sign up, sign in, sign out, 
session management, automatic token refresh, and user profile management.

**GlobalDoc** — Document database client. Read, write, delete, and query 
documents across collections with full TypeScript types.

**GlobalVault** — File storage client. Upload, download, list, and delete 
files with CDN delivery out of the box.

### Installation
```bash
npm install @globio/sdk
```

### Quick start
```typescript
import { Globio } from '@globio/sdk';

const globio = new Globio({ apiKey: 'glo_client_...' });

// Auth
await globio.id.signUp({ email, password });
await globio.id.signIn({ email, password });

// Documents
await globio.doc.set('scores', userId, { score: 0 });
const score = await globio.doc.get('scores', userId);

// Files
await globio.vault.upload(file, 'avatars/player.png');
```

### Notes

- Zero dependencies — uses native `fetch` and Web APIs only
- Works in browsers, Node.js 18+, Deno, Bun, and Cloudflare Workers
- Full TypeScript support with complete type definitions
- Available on both [npm](https://npmjs.com/package/@globio/sdk) and [JSR](https://jsr.io/@globio/sdk)
